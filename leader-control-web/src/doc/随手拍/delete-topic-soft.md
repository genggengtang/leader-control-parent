## 软删除随手拍，即只更新主题状态为查询不可见
#### 需添加授权请求头

## 接口地址 /delete-topic-soft/{id}
#### DELETE

## 请求参数
* {id} 随手拍主题编号 int

## 返回结果
* errorCode	返回码，0为成功，其余为错误码 int
* errorMsg 提示信息 string

## 返回例子
<pre>
{
	"errorCode":0,
	"errorMsg":"删除随手拍成功！"
}
</pre>

## 错误码
* -324 您删除的随手拍主题不存在！
* -325 您无权删除该随手拍主题！
* -326 该随手拍主题已被删除，不能重复删除！
* -327 删除随手拍主题失败！
