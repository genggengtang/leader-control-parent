## 添加项目关注
#### 需添加授权请求头 2018-3-20
#### 添加请求参数prjType 2018-4-17

## 接口地址 /prj-favorite-add
#### POST

## 请求参数
* prjId 项目编号，非空 int
* prjType 项目类型，参见字典表214，默认21401 int

## 返回结果
* errorCode	返回码，0为成功，其余为错误码 int
* errorMsg 提示信息 string
* data 返回数据，为关注编号 int

## 返回例子
<pre>
{
	"errorCode":0,
	"errorMsg":""，
	"data":1001
}
</pre>

## 错误码
* -203 您查询的项目编号不存在！
* -204 您查询的用户编号不存在！
* -205 您已关注该项目，不能重复关注！
* -206 项目关注失败！
* -207 不能关注自己负责的项目！
