## 解散聊天群组
#### 需添加授权请求头

## 接口地址 /my-cg-remove/{id}
#### DELETE

## 请求参数
* {id} 群组编号，非空 int

## 返回结果 - 机构数组
* errorCode	返回码，0为成功，其余为错误码 int
* errorMsg 提示信息 string


## 返回例子
<pre>
{
	"errorCode":0,
	"errorMsg":"解散聊天群组成功！"
}
</pre>

## 错误码
* -303 权限不足！
* -304 该聊天群组可能已解散！
* -311 默认群组不能加减人，也不能解散！
* -312 解散群组失败！
